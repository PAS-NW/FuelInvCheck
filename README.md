# PAS Fuel Invoice Reconciliation

Streamlit app for reconciling Motia / fuel card invoice PDFs against Vehicles.xlsx.

## Files

- app.py
- requirements.txt
- pas_logo.png
- .streamlit/config.toml

## Matching

Invoice short card numbers are matched against the full Fuel Card value in Vehicles.xlsx. The Vehicles tab takes priority over Off Hire - Sold.
